# Group19_Event Management System

## Project Purpose
This project is an Event Management System that helps users create, manage, and track events.
It allows organizers to post events, users to register, and admins to monitor activity.
The goal is to provide a simple, user-friendly platform for event planning and participation.

## Features
- User login/registration
- Create and manage events
- Event listing and search
- Registration/attendance tracking
- Admin controls (approve/edit/remove events)
